# Event loop 事件循环

- JavaScript是单线程语言
- 先将同步代码压入执行栈中,将异步代码放到任务队列中,微任务放到微任务队列，宏任务放到宏任务队列,将同步代码执行完之后

- 事件循环会先把微任务队列执行清空,微任务队列清空后，进入宏任务队列，取宏任务队列的第一个项任务进行执行，执行完之后，查看微任务队列是否有任务，有的话，清空微任务队列。然后再执行宏任务队列，反复操作,直到所有队列任务执行完毕。

- 微任务和宏任务
  - 微任务
    - Promise.then
  - 宏任务
    - script (可以理解为外层同步代码)
    - setTimeout/setInterval
