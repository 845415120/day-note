## 水平垂直居中

### 定位



```
.parent {
      width: 500px;
      height: 500px;
      position: relative;
    }

    .child {
      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
      /* 使⽤css3的transform来实现 */
    }
```

### flex



```
.parent {
      display: flex;
      justify-content: center;
      align-items: center;
      width: 600px;
      height: 600px;
      background: yellow;
    }

    .child {
      color: #fff;
      width: 300px;
      height: 300px;
      background-color: blue;
    }
```
